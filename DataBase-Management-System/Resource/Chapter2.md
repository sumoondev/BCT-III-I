### 1. Description and Significance of Data Model.
- Data Model is collection of conceptual tools for describing data, data relationships, data semantics and consistency constraints.
- Every database and database management system is based on a particular database model.
- It provides a way to describe the design of a database at logical level.
- It consists of set of rules and standards to defines how a data is organized in a database.
- It also provides set of operation to manipulate data in database.
- There are three different data models:
    - Physical data model
    - Logical data model
    - Conceptual data model

### 2. Brief description of each type of data model.
- **Physical Data Model**
    - This model describes how data is stored in the computer by representing records formats, records ordering and access path.
    - This model describes the way that data are saved on storage media such as disks and tapes.
    - With the physical data model, it is possible to implement the database at the system level.
    - Analysts usually use this model to calculate the storage estimates.
    - Complex to implement.

- **Logical Data Model**
    - This model is used to specify overall logical structure of databse.
